EcoMovilidadMX

Proyecto desarrollado en C# como parte de la asignatura de .NET de la Universidad Abierta y a Distancia de México (UNADM).

Descripción

EcoMovilidadMX es una aplicación de consola que permite planificar eventos de movilidad urbana mediante el cálculo del consumo de batería, costo operativo y personal técnico requerido para un conjunto de bicicletas eléctricas.

Además, integra una biblioteca externa para obtener métricas adicionales relacionadas con la movilidad.

Funcionalidades

- Captura el nombre del supervisor.
- Captura el nombre del evento.
- Captura el número de bicicletas.
- Captura la duración del evento.
- Calcula el consumo total de batería.
- Calcula el costo operativo.
- Calcula el consumo con reserva energética del 10%.
- Calcula el número de técnicos requeridos.
- Calcula el consumo promedio por bicicleta.
- Verifica si la cantidad de bicicletas es múltiplo de 5.
- Utiliza una biblioteca externa para calcular:
  - Índice de demanda.
  - Nivel de desgaste estimado.
  - Carga operativa.
- Aplica una conversión implícita y una conversión explícita.
